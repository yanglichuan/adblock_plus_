Así es como se configura el proxy de la conexión Wi-Fi para Adblock Plus:

• Abre ajustes inalámbricos
• Mantén pulsada tu conexión de red hasta que aparezca el menú contextual
• Selecciona Modificar la red
• Habilita la casilla Mostrar las opciones avanzadas
• Busca Configuración de proxy abajo y establécela en Manual
• Configura el Nombre de host proxy como localhost y el Puerto proxy como %d

Hay instrucciones detalladas en adblockplus.org.
